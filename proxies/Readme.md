## Proxy

Los proxies, son un nuevo tipo de objetos que fueron introducidos en el estandar ES5, que permiten customizar el comportamiento de las operaciones basicas como asignacion de variables, invocaci√≤n de funciones, acceso a las propiedades y demas.

### Esta compuesto por 3 Elementos:

- El Target, el cual es el objeto que el proxy envuelve.
- El Handler, que es el objeto que contiene las trampas.
- La Trampa que es el metodo que provee acceso a las propiedades.
